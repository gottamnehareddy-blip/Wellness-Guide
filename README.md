# Wellness-Guide
The Wellness Guide website is a simple platform for managing self-care activities.It includes sections for physical health, mental wellness, and routines or hobbies.Users can track daily habits such as hydration, exercise, and sleep.The system allows users to record their mood and monitor mental well-being.Users can add and manage hobbies to encourage creativity and relaxation.A reminder system helps users stay consistent with their routines.User data such as trackers, hobbies, and reminders are stored using browser local storage.The website provides an interactive and user-friendly interface.The project demonstrates the practical use of HTML, CSS, and JavaScript in building a web application.Overall, the system supports users in maintaining balanced and healthy daily habits.
